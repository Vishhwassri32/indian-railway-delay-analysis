# indian-railway-delay-analysis
🚆 Indian Railway Delay Analysis

Ever waited at the platform wondering “Why is my train always late?” 🕒
This project dives into Indian Railway delays with interactive visualizations to uncover hidden patterns, trends, and surprising insights!

🎯 What’s Inside

📊 EDA Magic – Cleaned & analyzed real train delay data

🎨 Visual Treats – Heatmaps, histograms & comparisons

⚡ Interactive Dashboards – Plotly & Cufflinks for dynamic insights

🔎 Storytelling with Data – Patterns across routes, timings, and seasons

🛠️ Tech Stack

🐍 Python (Pandas, NumPy)

🎨 Seaborn + Matplotlib (classic visuals)

⚡ Plotly + Cufflinks (interactive fun)

🚉 Sample Insights

Which trains run fashionably late 🕶️

Do weekends = longer delays?

How season/weather impacts delay trends 🌧️☀️

Which routes are most reliable 🚀

📂 Dataset

Cleaned dataset: indian_railway_delay_data_clean.csv
Columns include: Train_id, Train_name, Train_no, Source, Destination, Date, Delay_min, Season, Run_frequency
